Tinkercad Arduino Electrónica

- Resumen -

En esta práctica se desarrolló un circuito empleando una placa de Arduino y el código proporcionado por el profesor en Tinkercad.

- Introducción -

El objetivo de la práctica es apreder a usar Arduino.

- Materiales -

Los materiales empleados para desarrollar la práctica son:

Protoboard

Placa de Arduino

Cátodo Display de 7 segmentos

LEDS Rojos

Resistencia de 330

Resistencias de 1k

Jumpers macho - macho

- Desarrollo -

En electrónica se cuenta con una basta cantidad de materiales para poder crear circuitos, entre la inmensa cantidad de componentes, se emplearán solo los componentes descritos en la sección de materiales.

Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores. Esta plataforma permite crear diferentes tipos de microordenadores de una sola placa a los que la comunidad de creadores puede darles diferentes tipos de uso [1]. El display de 7 segmentos es un dispositivo optoelectrónico que permite visualizar números del 0 al 9 [2].

- Construcción -

El proceso de elaboración fue sencillo, debido a que el profesor nos proporcionó el código que sería transferido a la placa de Arduino, nuestro trabajo era comprender los códigos y hacer que funcionaran sobre una protoboard.
